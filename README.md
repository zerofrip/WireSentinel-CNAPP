# WireSentinel-CNAPP

Cloud-Native Application Protection Platform (CNAPP) for the WireSentinel ecosystem.

## Components

- **CSPM** — Cloud Security Posture Management
- **CWPP** — Cloud Workload Protection Platform
- **Kubernetes** — Kubernetes security
- **Containers** — Container runtime security
- **IaC** — Infrastructure-as-Code scanning
- **Secrets** — Secret detection
- **Supply Chain** — Dependency and supply-chain risk
- **SBOM** — Software bill of materials
- **Vulnerabilities** — Vulnerability management
- **Attack Path** — Cloud attack path analysis
- **Compliance** — Compliance monitoring
